# 🔥 PYROVISION
### Forest Fire Prediction & Simulation System
AI/ML Hackathon — Problem Statement 3

## What it does
- Predicts next-day fire probability at 30m resolution
- Simulates fire spread at 1, 2, 3, 6, 12 hour intervals
- Serves predictions via interactive web dashboard
- Multilingual AI assistant explains results in plain language

## Sections
| Folder | What's inside |
|--------|--------------|
| `data/` | Raw and processed geospatial datasets |
| `pipeline/` | Data ingestion and processing scripts |
| `models/` | U-Net and LSTM training code |
| `simulation/` | Cellular Automata fire spread engine |
| `backend/` | FastAPI REST server |
| `frontend/` | React dashboard |
